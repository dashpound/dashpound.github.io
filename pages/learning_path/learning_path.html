Portfolio

Below are some sample projects that I've completed as part of my education, I look to be adding more projects soon.

Amazon Recommendation Engine

Build a suite of custom models to create content-based and collaborative based recommendation models utilizing extracted Amazon customer review data.

Solution utilized deep learning and natural language processing techniques to produce recommendations.

Utilized plotly's Dash for eda visualization.
Deployed dashboard on Heroku.

Pop Punk Song Lyrics Generator

I grew up with the former lead singer of a pop-punk banded called Real Friends . It seemed like it would be a fun challenging project to see if I couldn't write a song for him using AI.

Solution required building my own data set by creating a scrapy bot to extract lyrics data from AZlyrics and compile a corpus for training. This set was passed to a recurrent neural network and a generative deep learning model to try to replicate Dan's style.

Inspired by Mohammed Ma'amari's Taylor Swift lyrics generator

Grokking Algorithms - Benchmarking

Grokking Algorithms is a great book that uses comics to explain key concepts in computer science. This repository is a collection of jupyter workbooks where I benchmark various algorithms against each other while manually implementing them in python.
